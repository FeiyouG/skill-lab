# Getting Started

## Prerequisites

- Deno installed for workspace development.
- Node.js 18+ for docs tooling (VitePress).

## Run checks in the workspace

```bash
deno task check
deno task test
```

## Build and run the CLI

```bash
deno task cli:build
deno task cli:dev
```

## Run docs locally

```bash
deno task docs:dev
```

Build static docs output:

```bash
deno task docs:build
```
