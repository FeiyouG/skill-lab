# Skill Lab

Skill Lab is a Deno workspace with:

- `slab` CLI for discovering, installing, and analyzing skills.
- `@FeiyouG/skill-lab-analyzer` for permission and risk analysis.
- `@FeiyouG/skill-lab` (SkillReader) for reading skill repositories from local and remote sources.

## Start here

- [Getting Started](/guide/getting-started)
- [CLI Overview](/cli/overview)
- [Analyzer Overview](/analyzer/overview)
- [SkillReader Overview](/skillreader/overview)

## Workspace packages

- `packages/cli` - CLI executable (`slab`)
- `packages/analyzer` - analyzer library
- `packages/skillreader` - skill reading library
- `packages/shared` - shared API/domain types
