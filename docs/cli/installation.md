# CLI Installation

## Homebrew (recommended)

```bash
brew tap feiyoug/tap/slab
```

or

```bash
brew tap feiyoug/tap
brew install slab
```

Verify installation:

```bash
slab --help
```

## Build from source

```bash
deno task cli:dev
```

## Build executable

```bash
deno task cli:build
```

## Install command locally

```bash
deno task cli:install
```
