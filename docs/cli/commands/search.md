# `slab search`

Search for skills in the registry.

## Usage

```bash
slab search <query> [--limit <n>] [--page <n>]
```

## Options

- `-l, --limit <n>`: number of results to show (default: `20`)
- `-p, --page <n>`: result page number (default: `1`)

## Examples

```bash
slab search code
slab search git --limit 10 --page 2
```
