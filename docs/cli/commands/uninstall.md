# `slab uninstall`

Remove an installed skill and its local tracking entry.

## Usage

```bash
slab uninstall <skill> [--global] [--yes]
```

## Options

- `-g, --global`: uninstall from global location
- `-y, --yes`: skip uninstall confirmation

## Examples

```bash
slab uninstall git-release
slab uninstall git-release --global --yes
```
