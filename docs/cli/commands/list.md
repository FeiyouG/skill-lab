# `slab list`

List locally tracked installed skills.

## Usage

```bash
slab list [--global] [--local] [--json]
```

## Options

- `-g, --global`: show only globally installed skills
- `-l, --local`: show only project-local installs
- `--json`: print structured JSON output

## Examples

```bash
slab list
slab list --global
slab list --json
```
