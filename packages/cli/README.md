# slab CLI

`slab` is the Skill Lab command line tool.

## Run

```bash
deno task --cwd packages/cli dev
```

## Build executable

```bash
deno task --cwd packages/cli build
```

## Install command

```bash
deno task --cwd packages/cli install
```

## Common commands

```bash
slab search <query>
slab info <skill>
slab install <skill>
slab list
slab update
slab uninstall <skill>
slab config
slab analyze <path>
```
